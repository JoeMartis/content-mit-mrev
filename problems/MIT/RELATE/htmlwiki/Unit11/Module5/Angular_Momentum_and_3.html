Angular Momentum and Angular Impulse Model

Compatible Systems

The system can be composed of any number of rigid bodies and point particles.

Relevant Interactions

Only external torques need to be considered. Internal torques do not change the system's angular momentum.

Law of change

The angular momentum of a system about a given point Q changes with time as:

\[\frac{d\vec{L}_Q}{dt}^{system}=\sum\vec\tau_Q^{external}\]

The integral form of the law of change relates the angular momentum of the system at an initial instant of time ti with the angular momentum of the system at a final instant of time tf as:

\[\vec{L}^{system}_Q(t_f)= \vec{L}^{system}_Q(t_i)+\int_{t_i}^{t_f} \sum\vec{\tau}_Q^{ext}dt\]

Angular Impulse

The time integral is called angular impulse:

\[\mathcal{J}_{Q} = \int_{t_{i}}^{t_{f}} \sum \vec{\tau}^{external}_{Q} dt\]

Constant Angular Momentum (zero external torque)

If the net external torque about a given point Q on a system is zero then the angular momentum of the system about that given point is constant:

\[\vec{L}^{system}_Q(t_f) = \vec{L}^{system}_Q(t_i)\]

Note 1:

As with the linear momentum of a system, the angular momentum of the system remains constant during the time interval ti < t < tf, but the angular momentum of the system constituents can change during that time.

Note 2:

The angular momentum of a system is constant only with respect to the point for which the net torque is zero.
