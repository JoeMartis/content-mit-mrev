Angular Momentum and Torque

Learning Goals

After working through this module, you should be able to

  • Explain why torque changes the angular momentum of a point particle or a rigid object
  • Decide if the angular momentum of a system of interacting objects remains constant
  • Apply the short time collision approximation

In the unit on momentum we have seen that an external force applied to a system changes the linear momentum of the system. In this unit we will learn that an external torque about a point changes the angular momentum of a system about the same point.

Linear Momentum is changed by External Forces Angular Momentum is changed by External Torques
\[\frac{d\vec{p}}{dt} = \sum\vec{F}^{ext}\]
\[\frac{d\vec{L}_Q}{dt} = \sum\vec{\tau}_Q^{ext}\]

The Angular Momentum and Torque of a point particle

Let's consider a particle of mass m moving with velocity \(\vec{v}\). The rate of change of the angular momentum vector about a given point Q is equal to the sum of the torques about point Q due to the forces acting on the particle :


\[\frac{d\vec{L}_Q}{dt} = \sum\vec{\tau}_{Q}\]


Derivation [show]
We will show here the differential form of the change of the angular momentum:

\[\frac{d\vec{L}_Q}{dt} = \sum\vec{\tau}_{Q}\]

The angular momentum of the particle about point Q is given by:

\[\vec{L}_{Q}=\vec{r}_{Q}\times\vec{p}=\vec{r}_{Q}\times m\vec{v}\]

where \(\vec{r}_Q\) is the particle's position vector measured from point Q. Taking the derivative with respect to time of the above expression we obtain:

\[\frac{d\vec{L}_{Q}}{dt}=\frac{d\vec{r}_{Q}}{dt}\times m\vec{v} + \vec{r}_{Q}\times m\frac{d\vec{v}}{dt}\]      (eq. 1)

Because the rate of change of the position vector is the velocity vector, the first term of (eq. 1):

\[\frac{d\vec{r}_{Q}}{dt}\times m\vec{v} = \vec{v} \times m\vec{v} = 0\]


Then, (eq. 1) becomes:

\[\frac{d\vec{L}_{Q}}{dt}= \vec{r}_Q\times m\frac{d\vec{v}}{dt} = \vec{r}_Q\times m\vec{a} = \vec{r}_Q\times \sum_{i=1}^n\vec{F}_i\]     (eq. 2)

where we have used Newton's 2nd law to substitute for the acceleration.

Because the forces are acting on a point particle, the position vector of the point particle, \(\vec{r}_Q\), is the position vector of the point of application of all the forces acting on the particle. As a result, the vector \(\vec{r}_Q\) is independent of the subscript i and can be put inside the sum of (eq. 2) to obtain:

\[\frac{d\vec{L}_{Q}}{dt}= \sum_{i=1}^n(\vec{r}_Q\times\vec{F}_i) = \sum_{i=1}^n\vec{\tau}_{Q,i}\]

The Angular Momentum and Torque of a System of N Point Particles

Let's consider a system formed by N point particles. The rate of change of the angular momentum of the system about a given point Q is equal to the sum of the external torques acting on the system about that point.

\[\frac{d\vec{L}_Q}{dt}^{system} = \sum\vec{\tau}_Q^{external}\]

1. In the expression above the angular momentum of the system is the sum of the angular momentum of each of the N particles composing the system:

\[\vec{L}_Q^{system} = \sum_{i=1}^N\vec{r}_{Q,i}\times m_i\vec{v}_i\]

2. The torques that will produce a change in the angular momentum are only the external torques on the system. The possible internal interactions between particles in the system will cause internal forces. It can be shown that the torque due to the internal forces cancel in pairs (as long as the 3rd law pairs are along the line joining both particles).

Angular Momentum and Torque of a Rigid Object

The result presented for a system of N particles is also valid for a rigid object. The rate of change of the angular momentum of a rigid object about a given point Q is equal to the net external torque on the object about point Q:

\[\frac{d\vec{L}_Q}{dt}^{object} = \sum\vec{\tau}_Q^{external}\]

where now \(\vec{L}_Q^{object}\) is the angular momentum of the rigid object with respect to a point 'Q.

This is a general result applied to objects of arbitrary shape rotating about an arbitrary axis, or translating and rotating.

In this course we will limit the application to the type of examples as those described in Module 2 and Module 3 where

  • the rotation is about an axis that does not change direction in space,
  • symmetrical objects are rotating about an axis of symmetry
  • planar objects are rotating about an axis perpendicular to the plane where the motion of the object is contained.
