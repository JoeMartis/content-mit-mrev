When to Choose Two Independent Systems Instead of One Combined System

Whenever you have a physical situation where two or more objects interact and, as a result, are constrained to move with the same acceleration, you can consider either the combined system or a set of sub-systems (for example, single-object systems). We will use the combined system when we are interested in external forces or in the acceleration of the combined system. However, if we need to obtain information about an internal force of the combined system, we must choose a sub-system for which that particular force is external. Sometimes you will have to work with the combined system and with single-object systems simultaneously.

Module Summary

Anything that limits or determines the motion of one or more objects is a constraint (on the motion). In consequence, there must be associated constraint forces that satisfy Newton's Second Law. In this module and in the next one, we consider systems where the constraint forces make all the elements in the system move with the same acceleration (in magnitude and direction).
