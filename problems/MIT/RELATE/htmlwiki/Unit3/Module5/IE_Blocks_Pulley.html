Illustrative Example: Two blocks and an ideal pulley.

Consider two blocks connected by an ideal rope that passes over an ideal pulley fixed on a table as shown in the figure. The blocks have masses m1 and m2. The block of mass m1 moves to the right along the horizontal and frictionless surface of the table while the block of mass m2 moves down. The goal of the problem is to find the acceleration of the blocks and the force of tension in the rope.

The blocks do not move in the same direction: block 1 moves to the right, and block 2 move down. Therefore the two accelerations are not equal and we cannot use the combined system approach we used in the examples of the previous module. We can treat a multiple-object system as a single point particle with total mass equal to the sum of the masses of its constituents ONLY when all the objects in the system are moving with the same acceleration. In this example we can't use this approach, so we will solve the problem using separate systems as indicated in the "standard approach" of applying Newton's Laws:

  • If we assume that the pulley is ideal then the force of tension at any point in the rope has the same magnitude, T. Therefore, the magnitude of the force of tension on block 1, T1R = T and the magnitude of the forces of tension on block 2, T2R = T.
  • The magnitude of the gravitational force on block 1, F1E = m1g, and the magnitude of the gravitational force on block 2, F2E = m2g.
4. Select a convenient coordinate system [show]
To obtain the blocks' acceleration we need to write Newton's second law for each block. For that purpose we need to select the coordinate system that we will use to write these equations. In the examples we have seen so far, the different objects were moving in the same direction, therefore we could use the same coordinate system for all of them.

In this problem we have that block 1 moves horizontally and block 2 vertically. We could use the same coordinate system to describe the forces on both blocks or we could use a different coordinate system for each block. We will show two solutions to this problem.

First solution. Use the same coordinate system for both blocks where the +x - axis is to the right and the +y axis is up. In this coordinate system, the direction of the acceleration of block 1 is along the +x and the direction of the acceleration of block 2 is along the -y - axis.

Second solution. Use different coordinate systems for each block. We will choose each coordinates system such that the acceleration of each box is along the positive direction of one of the axis.

For block 1 we will use +x to the right and +y pointing up (the acceleration of block 1 is along the +x axis).
For block 2 we will use +y - axis pointing down and the +x axis to the right (the acceleration of block 2 is along the +y axis).

6. Related Accelerations Constraint [show]
In this particular example it is easy to find the relationship between the blocks. Because the rope does not stretch or becomes slack, its length has to be constant during the motion of the blocks, therefore when block 2 goes down 1 cm, block 1 moves to the right the same distance. As a result, the acceleration of the two blocks have the same magnitude. If we call the magnitude of the acceleration 'a', then the Related Acceleration Constrained is given by:
Equal magnitudes
\(|\vec{a}_{1}| = |\vec{a}_{2}| = a \)
Different directions
\(\vec{a}_{1}\)     points to the right
\(\vec{a}_{2}\)     points down
