Point Particles, System vs. Environment and Internal vs. External Forces

It is possible to use Newton's Laws to study the motion of a collection of individual bodies all at once. The bodies whose motion is being simultaneously described is the "system" of interest. Any objects that interact with the system constitute the "environment".

Learning Goals

After completing this module you should be able to:

  • Define the term point particle and explain when to apply the point particle approximation
  • Define the terms system and environment
  • Define internal and external forces
  • Identify the internal and external interactions for a given situation.

Point Particle Approximation

The next unit is devoted to the description of translational motion, which begins with defining the position of an object measured relative to a point of reference. In the case of translational motion of a solid non-rotating object, all the points in the object are moving in the same way and any point in the object can be used to represent the position of that object. (This is not the case if the object is rotating). In other words, the dimensions and shape of the object are not relevant for describing its translational motion, and we can consider the object to be represented by a single point in space.

The point particle approximation is a good representation of a rigid object in which rotation is ignored. In this approximation the object's mass is considered to be localized at the object's center of mass (to be defined later), which is conventionally used as the "position" of an extended object.

System, System Constituents, and Environment, Internal and External Forces

A system is the object or the group of objects whose motion we are describing. In a full treatment of mechanics, allowable massive objects include point particles, rigid bodies, elastic bodies, fluids, and gasses. In this course, we will consider only point particles and rigid bodies as possible system constituents. Systems can have massless objects in them also, like ideal ropes and springs.

Once the system is specified, all objects that are outside of its boundary are defined as the environment. Of course, objects in the environment can interact with the objects inside the system and affect their state of motion. The forces resulting from the interaction between the objects in the environment and those inside the system are called external forces.

Forces resulting from the interactions between two objects within the system are called internal forces. In the unit on momentum, you will learn that these affect neither the momentum of the system nor the motion of its center of mass.

Free Body Diagrams show the system isolated from its environment. Only external forces are shown in the free body diagram.
