Multiple Systems and Constraints

Often it is desirable to consider the relative motion of two objects. (A particularly common problem is to determine when two objects meet.) Because the Dynamics and Net Force Model and its special cases are restricted to describe only a single point particle, the most common approach to tracking the motion of two objects is to treat them as separate systems and apply the appropriate model to each. These descriptions must however use the same time coordinate.

Learning Goals

After completing this module, you should be able to:

  • Solve motion problems that deal with the motion of more than one object.
  • Solve problems involving more than one concept.

Tracking Two Objects

The separate models that are applied to each object will be related to each other first by using the same time coordinate and second by constraints imposed by the specific motions of the two systems (e.g. the player has to be at the same location as the ball in order to catch it). The basic elements of the approach are summarized in this table:

Steps for Approaching a Problem Involving Motion of Two Objects
  1. Set up a single coordinate system that is useful to describe the motion of both systems.
  2. Determine the model that applies to each system independently.
  3. Constrain the models using what you know about the motion of the two systems (e.g. where are they constrained to start, meet, etc.).
