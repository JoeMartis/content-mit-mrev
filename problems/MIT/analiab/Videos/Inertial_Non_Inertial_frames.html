Inertial and Non-Inertial Frame of References

A frame of reference is a coordinate system (e.g. an orthogonal set of axes) relative to which we describe the motion of an object. If the motion of the object relative to a given frame of reference obeys Newton's first law, then the frame of reference is called an inertial frame of reference. All inertial frames of references are moving at a constant speed with respect to each other. A frame of reference in which Newton's first law does not hold is called a non-inertial frame of reference. An accelerating frame of reference is non-inertial.
