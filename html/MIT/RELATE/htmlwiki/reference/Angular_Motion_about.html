Restrictions to the Angular Velocity and Angular Acceleration about a Single Axis Model

Angular Motion about a Single Axis with Constant Angular Acceleration is a sub-model of the Angular Velocity and Angular Acceleration about a Single Axis Model. This model applies to a rigid body constrained to rotate about a single axis and experiencing an angular acceleration which is constant in size and direction.

Mathematical Representations

\(\mathcal{}\omega_{a}(t) = \omega_{i,a} + \alpha_{a}(t-t_{i})\)

\(\theta_{a}(t) = \theta_{i,a} + \omega_{i,a}(t-t_{i})+\frac{1}{2}\alpha_{a}(t-t_{i})^{2}\)

\(\omega_{f,a}^{2} = \omega_{i,a}^{2} + 2\alpha_{a}(\theta_{f,a}-\theta_{i,a})\)

\(\theta_{f,a} = \theta_{i,a} + \frac{1}{2}(\omega_{i,a}+\omega_{f,a})(t_{f}-t_{i})\)
